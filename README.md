# Data Deletion Request Page - JPtech

This repository hosts the official Data Deletion Request page for apps developed by **JPtech**, as required by the [Google Play Data Safety Policy](https://support.google.com/googleplay/android-developer/answer/10787469).

## Purpose
Users of our mobile applications can visit the live page to request the deletion of their account and associated personal data.

## Live Link
The page is hosted via GitHub Pages at:
[https://jptechntr.github.io/data-deletion-request/] (https://jptechntr.github.io/data-delete-requests/)

## Technical Details
- **Framework:** Simple HTML/CSS
- **Form Handling:** Google Forms (Embedded)
- **Contact Email:** vikkiswm9885@gmail.com

## How to use
If you are a user looking to delete your data, please follow the link above and fill out the provided form. Our team will process the request within 7 business days.
